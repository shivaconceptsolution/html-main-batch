List Tag:-

It is used to create list item using ordered list, unorderd list and defination list.

1) Ordered List
ol means orderd list and li means list item.

<ol>
Courses
<li>Basic Course</li>
<li>Advance Course</li>
<li>Designing Course</li>
</ol>

2) Unorderd List:-
  it contain set of list items without any sequence.
  <ul>
    Title
    <li> Content....</li>
    ...
  </ul>

3) Defination List:-
   It contain set of list items to define data.
   dl means defination list, dt means defination title and dd means defination description.
   <dl>
       <dt>  <dd>   </dd> </dt>
       
    </dl>
